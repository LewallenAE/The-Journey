The Box Model In A Nutshell

|<----------- margin ----------->|
┌───────────────────────────────┐
│         border (outer)        │
│   ┌───────────────────────┐   │
│   │     padding (space)   │   │
│   │   ┌───────────────┐   │   │
│   │   │   content     │   │   │
│   │   └───────────────┘   │   │
│   └───────────────────────┘   │
└───────────────────────────────┘
        content is the content
        padding pushes content inward
        border is the border around the content
        margin pushes everything else away
